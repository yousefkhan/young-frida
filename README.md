# young-frida

Simple app: Express API serves mock transactions, and a single page shows them in a table.

## Run locally

```bash
cd server
npm install
node server.js
```

Open http://localhost:3000 in your browser.

## Project layout

- `server/` – Express app and mock data
- `public/` – HTML, CSS, JS for the table
- `CHECKLIST.md` – Tasks (2 days)
